## Description
Run k6 load tests against the Medusa store products/collections endpoints and the Payload CMS homepage global API to verify they both meet the p95 < 200ms response time target under realistic concurrent traffic. Homepage data is fetched on every ISR cache miss and must remain fast to prevent user-visible delays during traffic spikes.
